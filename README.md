# dockerfiles
Docker files built by ssfdre38 to be used for AMP (Application Managment Panel) By Cubecoders. All of this is for testing only and maybe to be added into the Cubecoders ampbase if works. All docker containers are at https://hub.docker.com/r/ssfdre38/ampdocker and can be used as a replacement for cubecoders/ampbase.

All Docker files are use at your own risk and do not contain support from CubeCoders at all.
